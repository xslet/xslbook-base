About xslbook-base

This library implements templates and parameters that are imported and commonly used by all other xslbook libraries, along with xslbook-do.

The match templates applied to elements are used for body texts of book, chapters, and block elements. Book texts and chapter texts are not used in XML data operation elements (those are implemented in xslbook-do), but texts of block elements are used in XML data operation elements. (Block elements could appear in or elements.) Therefore match templates for elements are implemented separatedly by @mode attribute depending on whether it is used in XML data operation elements. The @mode of the template for book texts and chapter texts has the value "xslbook", and the @mode of the template for block elements has no value.

book:xsl_url and book:xsl_dir are common parameters for a XSL file specified at a processing instruction. The former is a path of the XSL file, and the latter is a parent directory of it. book:xsl_dir is also used for getting paths of resource files in the same directory, e.g. xslbook.css and xslbook.js.

In addition, this library provides more named-templates.

Usage

This section explains how to import xslbook-base and call its templates in your XSL application or library.

Import xslbook-base

To import xslbook-base.xsl into another XSL file, the following handworks are needed. Since xslbook-base depends on some xslet libraries, they are needed to import, too.

  1. Add the namespace declarations of xslbook and xslutil in a XSL file.

    <xsl-stylesheet version="1.0"
      xmlns:xsl="http://www.w3.org/1999/XSL/Transform"
      xmlns:ut="https://github.com/xslet/2020/xslutil"
      xmlns:book="https://github.com/xslet/2020/xslbook"> <!-- This! -->
      ...

  2. Add a import element of xslbook-base in the XSL file.

      ...
      <!-- dependency XSL files are already imported in xslbook-base.xsl -->
      <xsl:import href="./xslbook-base.xsl"/>
      ...
